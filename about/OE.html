On-Premise Infrastructure

Most of my work has been on Linux, whether managing workstations or servers. My responsibilities included:

- Setting up and managing Linux servers and workstations for development and production environments.
- Configuring Windows servers and workstations to support cross-platform development and integration.
- Deploying and maintaining GitLab Enterprise Edition to facilitate source code management and CI/CD workflows.
- Setting up and managing Jenkins for CI/CD to automate software build, test, and deployment processes.
- Setting up GitLab and Nexus container registries to efficiently manage and distribute container images.
- Administering databases, including PostgreSQL and MicrosoftSQL ensuring high availability and performance.
- Configuring autoscaling Linux virtual machines to optimize resource utilization and performance.
- Setting up and managing Kubernetes clusters for container orchestration, scalability, and efficient resource management.
- Setting up networking and UFW rules to enforce security, access control, and firewall configurations.
- Deploying containerized web applications using Docker and Kubernetes for scalable and efficient deployment.
- Implementing monitoring solutions with tools like Grafana Stack and Zabbix to ensure system health and performance tracking.
- Setting up and managing notification systems using NTFY and Uptime Kuma to provide real-time alerts and status monitoring.
