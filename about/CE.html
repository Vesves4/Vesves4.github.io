Cloud Infrastructure

Azure

Two of my largest projects leveraged Azure as their cloud infrastructure, both relying on scalable virtual machines to power their CI/CD pipelines. My responsibilities in these projects included:

- Deploying and managing infrastructure to ensure reliability and scalability.
- Configuring autoscaling virtual machines to optimize resource usage and performance.
- Setting up networking and RBAC roles to enforce security and access control across different development teams.
- Managing storage containers to efficiently handle data storage and retrieval.
- Organizing resource groups for streamlined resource management and cost optimization.
- Administering databases, including PostgreSQL and Azure SQL, ensuring high availability and performance.
- Configuring and managing Databricks, enabling advanced data analytics and machine learning workflows.
- Deploying web applications using Azure App Services and containerized solutions.
- Implementing serverless architectures by deploying Azure Function Apps to streamline event-driven workloads.

GCP

My early projects leveraged Google Cloud Platform (GCP) as their cloud infrastructure, utilizing GKE, domains, and web applications. My responsibilities in these projects included:

- Deploying and managing infrastructure to ensure reliability, scalability, and security.
- Setting up and maintaining GKE clusters and containerized applications to enable seamless orchestration and scalability.
- Deploying web applications using GCP App Engine, Kubernetes, and containerized solutions for optimized performance and flexibility.

Openshift

I have limited experience in this area, but I have worked on deploying applications both manually and through automated pipelines using Jenkins. Additionally, I have configured ConfigMaps and mounted volumes to manage application settings and dependencies efficiently.
